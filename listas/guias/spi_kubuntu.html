Guía de configuración de acceso al Sistema de Patentes

Pasos para configurar en un puesto stand alone al acceso de Sistema de Patentes

Para poder acceder al Sistema de Patente de la Provicia desde un puesto Linux Stand Alone es necesario realizar los siguientes pasos:

Instalar paquetes necesarios:

Para instalar el Terminal de patente:

sudo apt-get install x3270

Para instalar las librerías necesarias para poder asignar la Romu necesaria de impresión:

sudo apt-get install pr3287   tk8.5

Descargar archivos de configuración:

Descargar el archivo Archivos necesarios para Acceso SPI el cual contiene los íconos del Acceso SPI e ImprePat1, y los archivos de configuración necesarios.

Descomprimir el archivo tar.gz

  1. Mover los archivos xpr3287 y xpr3287.tcl a la carpeta /usr/local/bin
  2. Primero Editar archivo .x3287pro, luego mover los archivos x3287conectar y x3287desconectar .x3270pro a la carpeta /home/$USER Obs.
    Para mover el archivo .x3270pro es necesario activar la opción "Mostrar íconos ocultos" del la barra de herramientas "Ver".
  3. Mover los Iconos "Acceso SPI" e "ImprePat1" al Desktop

Obs. Para mover los archivos es necesario ser administrador (sudo)

Editar archivos de configuración:

Además de copiar los archivos de configuración es necesario modificarlos para adaptarlos a la configuración personal de cada usuario.

Editar archivo .x3270pro

  1. Click derecho sobre el archivo
  2. Abrir con ...
  3. Escribir en la barra : kwrite (o cualquier otro editor de texto simple)
  4. Donde dice "nombre_logico_impresora" poner el nombre lógico de la impresora que corresponda, guardar y cerrar.
    Obs. Esta impresora es la que va a quedar por defecto para las impresiones de pantalla (Print Screen).

Ej. El archivo tiene que quedar de la siguiente manera:

! x3270 profile 
x3270.emulatorFont: -misc-fixed-medium-r-normal--20-200-75-75-c-100-iso8859-1
x3270.printTextCommand: lpr -o cpi=12 -o page-top=25 -o page-left=20 -P hpXXXX
x3270.keypadOn: true
x3270.menuBar:  true

Editar el icono "Acceso SPI"

  1. Click derecho al ícono
  2. Propiedades
  3. Aplicación
  4. Orden : x3270 -reconnect -model 3279-2 "completar_ROMU"@10.1.4.11
    Donde dice "completar_ROMU" reemplazar por la ROMU necesaria
    (ej. x3270 -reconnect -model 3279-2 ROMUxxxx@10.1.4.11)

Editar el ícono "ImprePat1"

  1. Click derecho al ícono
  2. Propiedades
  3. Aplicación
  4. Orden : /usr/local/bin/xpr3287 "nombre_logico_impresora" "completar_ROMU"@10.1.4.11
    Donde dice "nombre_logico_impresora" reemplazar con el nombre lógico de la impresora
    Donde dice "completar_ROMU" reemplazar con la ROMU necesario
    (ej. /usr/local/bin/xpr3287 hpXXXX ROMUxxxx@10.1.4.11)
